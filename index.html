JavaScript Event Loop Trainer

Интерактивный тренажёр, который помогает понять порядок выполнения JavaScript-кода: синхронные вызовы, микрозадачи, таймеры и async/await.

Как тренироваться

  1. Выберите сложность и тему задачи.
  2. Предскажите порядок значений из console.log.
  3. Проверьте ответ и изучите пошаговую визуализацию Call Stack, Microtask Queue и Task Queue.

Темы

Promise.then, Promise.finally, queueMicrotask, setTimeout, async/await, вложенные колбэки и очереди задач.

Practice JavaScript event loop execution order with synchronous code, microtasks, timers, Promises, and async/await.
